공유 캘린더 웹페이지 사용 설명서

이 문서는 구글 캘린더와 연동되는 공유 캘린더 웹페이지의 사용 방법을 안내합니다.

1. 시작하기

1.1. 최초 로그인 및 권한 동의

  1. 웹페이지에 접속하면 오른쪽 위에 [Google 계정으로 로그인] 버튼이 보입니다. 이 버튼을 클릭하세요.
  2. 구글 계정 선택 창이 나타나면, 연동할 본인의 구글 계정을 선택합니다.
  3. 다음으로, "이 앱이 Google 계정에서 다음 작업을 할 수 있도록 허용하시겠습니까?"라는 권한 동의 화면이 나타납니다. 이 앱은 캘린더의 일정을 읽고 쓰는 기능이 필요하므로, [모두 선택] 후 [계속] 버튼을 눌러 권한을 허용해야 모든 기능을 정상적으로 사용할 수 있습니다.

2. 화면 구성

로그인하면 다음과 같은 화면이 나타납니다.

  • ① 캘린더: 중앙에 위치한 달력입니다. 구글 캘린더에 등록된 일정이 자동으로 표시됩니다.
  • ② 날짜 이동: 헤더의 < > 버튼을 눌러 이전 달, 다음 달로 이동할 수 있습니다.
  • ③ 새 일정: [+ 새 일정] 버튼을 눌러 새로운 일정을 등록할 수 있습니다.
  • ④ 작성자 필터: 공유 캘린더에 일정을 등록한 모든 사람의 목록이 나타납니다. 특정 사람을 선택하면 그 사람이 작성한 일정만 필터링해서 볼 수 있습니다.
  • ⑤ 통계 위젯: 오른쪽 사이드바에는 '연락 현황'과 '휴가 현황'이 월별/연간 단위로 자동 집계되어 표시됩니다.

3. 주요 기능

3.1. 새 일정 추가하기

  1. 헤더의 [+ 새 일정] 버튼을 클릭하면 일정 추가 팝업창이 나타납니다.
  2. 제목, 날짜, 설명을 입력합니다.
  3. 나만 보기 (비공개): 이 옵션을 체크하면, 캘린더를 공유하는 다른 사람들에게는 이 일정이 보이지 않게 비공개로 등록됩니다.
  4. [저장] 버튼을 누르면 일정이 실제 구글 캘린더에 저장되고, 잠시 후 웹페이지 화면에도 자동으로 반영됩니다.

3.2. 작성자별 일정 필터링

  1. 헤더의 '모든 작성자' 라고 표시된 드롭다운 메뉴를 클릭합니다.
  2. 캘린더를 공유하는 팀원들의 목록이 나타납니다.
  3. 특정 팀원의 이름을 선택하면, 달력과 오른쪽 통계 위젯에 해당 팀원이 작성한 일정만 필터링되어 표시됩니다. 다시 전체 일정을 보려면 '모든 작성자'를 선택하세요.

3.3. 통계 위젯 활용하기

오른쪽의 통계 위젯은 현재 보고 있는 달과 올해 전체를 기준으로 자동으로 집계됩니다.

  • 연락 현황: 일정 제목에 'OOO 통화', 'XXX 전화', 'ㅁㅁㅁ 연락' 과 같이 이름과 특정 키워드가 함께 포함된 경우, 해당 이름의 연락 횟수를 자동으로 카운트합니다.
  • 휴가 현황: 일정 제목에 '휴가', '연차', '반차' 라는 단어가 포함된 경우, 작성자별로 각 항목의 사용 횟수를 자동으로 카운트하여 보여줍니다.

4. 문제 해결

  • 로그인 후 일정이 보이지 않을 때: Render의 무료 서버는 15분간 사용이 없으면 잠시 '잠자기 모드'로 전환됩니다. 이 경우, 로그인 후 약 30초 정도 기다렸다가 페이지를 새로고침하면 서버가 다시 깨어나면서 일정을 정상적으로 불러옵니다.
  • 새로 추가한 일정이 바로 보이지 않을 때: 구글 캘린더에 저장되고 다시 웹페이지에 반영되기까지 약간의 시간이 걸릴 수 있습니다. 잠시 후 페이지를 새로고침 해보세요.
